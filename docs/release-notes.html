Version 1.0.0.0

  • Initial Release
  • Automatically calculates total basic ingredients for all recipes added to the recipes list.  Ingredients are calculated dynamically as soon as you add recipes or adjust their quantities. 
  • Displays the breakdown of all crafted items needed to make the recipes added.
  • Allows you to copy the total ingredients section to the windows clipboard
  • Supports almost all crafted items from No Man's Sky.
  • Light and dark theme with various accent colors.
  • Recipes list can be filtered down to specific groupings.  

Version 1.1.0.0

  • Several recipes were added to allow calculation of repair costs for various technology items.
  • The Data Model was converted over to use LiteDB for data storage.  More info on this can be found on the Crafting Calculator Wiki
  • The Recipe Ingredients Needed tree view will now remember expanded nodes when the quantities are adjusted or new recipes are added.
  • Game descriptions for the Recipes and Basic Elements were included in the tooltips for Recipes and Ingredients.  Total Ingredients now shows a tooltip when you hover over the ingredients.
  • Total Ingredients list is now ordered alphabetically by Ingredient Name.
  • Recipes List can be saved to customized favorites so that you can easily reload the recipes you use frequently.
